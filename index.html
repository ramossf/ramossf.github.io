3-Column Layout

This is a template example that uses the Bootstrap framework to create a responsive three-column layout. The left column (narrow) contains a sidebar, the center column (mid) contains features with image/caption, and the right column (widest) contains content.

Affix

This layout utilizes the Bootstrap Affix component using data attributes. You can see the Affix plugin in action as you scroll down the page. The leftmost side is "pinned" until 140px is reached, and the middle column is un-pinned once 250px is reached. See the CSS .affix and .affix-top classes to see how this works.
